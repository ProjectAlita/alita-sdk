Fetches Azure DevOps work item hierarchy with readiness analysis and tree visualization. Returns JSON + TXT filenames.
**Input:** `{"id": 9, "include_dependencies": true, "include_links": true}`
**Parameters:**
- `id` (int, required): Feature/Epic work item ID
- `include_dependencies` (bool|int, default=true): Depth control. `false`=feature only, `true`=all levels, `1-N`=max level count
- `include_links` (bool, default=true): Extracts URLs from descriptions to categories: Figma, wiki, item, other.
**Output:** `["9_feature.json", "9_feature.txt"]`
**JSON Structure:** Feature + dependencies, readiness (completion %, status: AT_RISK/IN_PROGRESS/READY/COMPLETE, blockers, type breakdown), enriched links
**TXT Format:** Hierarchical tree, statuses.
