Fetches Azure DevOps work item hierarchies with all descendants recursively. Returns a list of lists with work item IDs. Each work item is fetched with complete hierarchy structure, including parent-child relationships.
**Required Input (JSON object):**
```json
{
  "ids": [9, 42, 123],
  "output_expand": true,
  "output_name": "my_run"
}
```
- `ids` (integers): List of work item IDs to process. Examples: `[9]`, `[9, 42, 456]`
- `output_expand` (boolean): Controls output content. `true` = include all descendants, `false` = only input IDs
- `output_name` (string, optional): Output identifier/postfix for file names. Auto-generated random 8-character string if not provided.
**Output Format Example:**
```json
[[9_feature, 10, 11, 12, 13], [42_feature]]
```
