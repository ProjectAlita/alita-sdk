## Data Flow

The data flow within `langraph_agent.py` revolves around the creation and manipulation of state graphs for managing conversational agents. The data originates from YAML schemas and user inputs, which are processed and transformed through various nodes and edges within the state graph. The data is stored in a `BaseStore` and manipulated using tools and functions defined in the schema. The flow of data is managed by classes like `ConditionalEdge`, `DecisionEdge`, and `TransitionalEdge`, which determine the next steps based on conditions, decisions, and transitions. The final output is generated by the `LangGraphAgentRunnable` class, which processes the input data and returns the result.

Example:
```python
class ConditionalEdge(Runnable):
    def invoke(self, state: Annotated[BaseStore, InjectedStore()], config: Optional[RunnableConfig] = None) -> str:
        input_data = {}
        for field in self.condition_inputs:
            input_data[field] = state.get(field, "")
        template = EvaluateTemplate(self.condition, input_data)
        result = template.evaluate()
        return result
```
In this example, the `ConditionalEdge` class processes the state data and evaluates a condition to determine the next step.

## Functional Flow

The functional flow of `langraph_agent.py` involves the following steps:
1. **Loading the Schema:** The YAML schema is loaded and parsed.
2. **Creating Nodes and Edges:** Nodes and edges are created based on the schema.
3. **Validating the Graph:** The graph is validated to ensure correctness.
4. **Preparing the Output Schema:** The output schema is prepared by setting up channels and attaching nodes and edges.
5. **Executing the Graph:** The graph is executed by processing input data and invoking the appropriate nodes and edges.
6. **Returning the Result:** The final result is returned, including the output and thread ID.

Example:
```python
def create_graph(
        client: Any, 
        yaml_schema: str, 
        tools: list[BaseTool], 
        *args,
        memory: Optional[Any] = None,
        store: Optional[BaseStore] = None,
        debug: bool = False,
        **kwargs
    ):
        schema = yaml.safe_load(yaml_schema)
        lg_builder = StateGraph(state_class)
        for node in schema['nodes']:
            node_type = node.get('type', 'function')
            node_id = clean_string(node['id'])
            tool_name = clean_string(node.get('tool', node_id))
            if node_type in ['function', 'tool', 'loop', 'loop_from_tool']:
                for tool in tools:
                    if tool.name == tool_name:
                        lg_builder.add_node(node_id, FunctionTool(tool=tool, name=node['id']))
        compiled = prepare_output_schema(lg_builder, memory, store, debug)
        return compiled.validate()
```
In this example, the `create_graph` function loads the schema, creates nodes, and prepares the output schema.
